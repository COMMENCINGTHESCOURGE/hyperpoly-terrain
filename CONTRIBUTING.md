# Contributing to MANIFOLD

Welcome to the Continuity Engine. We enforce mass conservation strictly.

## 🚀 Getting Started

If you are a new contributor, please check the issues labeled **`good first issue`**. These are specifically scoped to introduce you to the tensor dataflow and rendering pipeline without requiring deep distributed systems knowledge.

### Quick Start
1. Clone the repository.
2. Run `.\install-deps.ps1` (Windows) to fetch Filament and Vulkan.
3. Check the `.github/ISSUE_TEMPLATE` folder for guidance.

*Continuity is not a feature. It's the foundation.*
